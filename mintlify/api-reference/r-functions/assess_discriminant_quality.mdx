# Check if discriminant parameters meet quality criteria


 Based on analysis findings, check if discriminant parameters are likely
 to produce high-quality random numbers.


## Usage

```r

 assess_discriminant_quality(a, b, c, discriminant)

```

## Parameters

<ParamField body="a" type="any">
  Parameter a (should be small, 1-3 preferred)
</ParamField>
<ParamField body="b" type="any">
  Parameter b
</ParamField>
<ParamField body="c" type="any">
  Parameter c (moderately negative preferred)
</ParamField>
<ParamField body="discriminant" type="any">
  Discriminant value (1000-2000 range preferred)
</ParamField>

## Returns


 list with quality assessment
