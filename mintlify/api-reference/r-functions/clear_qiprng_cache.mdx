# Clear the cache


 Removes all cached results or specific cache entries.


## Usage

```r

 clear_qiprng_cache(pattern = NULL)

```

## Parameters

<ParamField body="pattern" type="any">
  Optional pattern to match specific cache entries
</ParamField>

## Returns


 Invisible NULL
