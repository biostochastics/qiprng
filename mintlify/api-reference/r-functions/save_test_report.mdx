# Save a comprehensive test report


 Save a comprehensive test report


## Usage

```r

 save_test_report(suite, filename = NULL)

```

## Parameters

<ParamField body="suite" type="any">
  The test suite with results
</ParamField>
<ParamField body="filename" type="any">
  Optional filename for the report
</ParamField>

## Returns


 The path to the saved report (invisibly)
