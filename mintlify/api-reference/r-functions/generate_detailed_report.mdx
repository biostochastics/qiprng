# Generate detailed report for top and bottom performers


 Generate detailed report for top and bottom performers


## Usage

```r

 generate_detailed_report(results, summary_data, n_top = 5, n_bottom = 5)

```

## Parameters

<ParamField body="results" type="any">
  List of test results
</ParamField>
<ParamField body="summary_data" type="any">
  Summary statistics data frame
</ParamField>
<ParamField body="n_top" type="any">
  Number of top performers to report
</ParamField>
<ParamField body="n_bottom" type="any">
  Number of bottom performers to report
</ParamField>

## Returns


 Character string with detailed report
