# Set cache TTL (Time To Live)


 Configure how long cached results should be kept before expiration.


## Usage

```r

 set_cache_ttl(hours = 24)

```

## Parameters

<ParamField body="hours" type="any">
  Number of hours before cache entries expire
</ParamField>

## Returns


 Invisible NULL
