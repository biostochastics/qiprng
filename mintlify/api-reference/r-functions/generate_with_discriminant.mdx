# Generate random numbers using specific discriminant parameters


 Creates a PRNG instance with the specified quadratic parameters and generates
 random numbers. The function uses cryptographic mixing for enhanced quality.


## Usage

```r

 generate_with_discriminant(a, b, c, n = 50000, precision = 256)

```

## Parameters

<ParamField body="a" type="any">
  Quadratic coefficient (must be > 0)
</ParamField>
<ParamField body="b" type="any">
  Linear coefficient
</ParamField>
<ParamField body="c" type="any">
  Constant term (must be < 0)
</ParamField>
<ParamField body="n" type="any">
  Number of samples to generate (default: 50000)
</ParamField>
<ParamField body="precision" type="any">
  MPFR precision in bits (default: 256)
</ParamField>

## Returns


 A numeric vector of n random numbers uniformly distributed on  0,1


## Details


 The function creates a PRNG using the quadratic recurrence relation:
 x_{n+1} = (a x_n^2 + b x_n + c) \mod 1

 Key configuration:

  Cryptographic mixing is enabled for better randomness quality
  Parallel filling is disabled to avoid performance issues
  The PRNG uses the specified MPFR precision for calculations



## Examples

```r


 # Generate random numbers with specific parameters
 samples <- generate_with_discriminant(a = 2, b = 5, c = -2, n = 10000)
 hist(samples, breaks = 50)


```
