# Benchmark large-scale random number generation


 This function focuses on benchmarking the performance of random number generators
 when generating large quantities of random numbers. It is particularly useful for
 evaluating the scaling behavior and memory efficiency of different generators.


## Usage

```r

 benchmark_large_generation(
   n_values = c(1e6, 5e6, 1e7, 5e7),
   configs = list(
     "default" = list(),
     "optimized" = list(buffer_size = 1e6)
   ),
   generators = list(
     "qiprng" = function(n) generatePRNG(n),
     "base_r" = function(n) runif(n)
   ),
   repetitions = 3
 )

```

## Parameters

<ParamField body="n_values" type="any">
  Vector of large sample sizes to benchmark
</ParamField>
<ParamField body="configs" type="any">
  List of QIPRNG configurations to test
</ParamField>
<ParamField body="generators" type="any">
  List of generator functions to compare
</ParamField>
<ParamField body="repetitions" type="any">
  Number of repetitions for each benchmark
</ParamField>

## Returns


 A list containing:

       results: Raw benchmark data
       summary: Summary statistics for each generator and configuration
       plots: A list of ggplot objects visualizing the results



## Examples

```r


 # Compare QIPRNG to base R for large sample sizes
 results <- benchmark_large_generation(
   n_values = c(1e6, 5e6),
   configs = list(
     "default" = list(),
     "large_buffer" = list(buffer_size = 100000)
   ),
   repetitions = 2
 )

 # View the results
 print(results$summary)
 print(results$plots$time_plot)

```
