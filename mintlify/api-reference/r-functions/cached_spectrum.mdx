# Cached spectrum function


 Cached wrapper for stats::spectrum() function. Uses R.cache to store
 results for repeated calls with the same data and parameters.


## Usage

```r

 cached_spectrum(x, ...)

```

## Parameters

<ParamField body="x" type="any">
  A univariate or multivariate time series
</ParamField>
<ParamField body="..." type="any">
  Additional arguments passed to spectrum()
</ParamField>

## Returns


 An object of class "spec"
