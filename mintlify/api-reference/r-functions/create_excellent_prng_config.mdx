# Create PRNG configuration using excellent discriminant


 Creates a complete PRNG configuration object from a single excellent
 discriminant parameter set. The configuration is optimized for high-quality
 random number generation.


## Usage

```r

 create_excellent_prng_config(
   discriminant_params,
   precision = 256,
   use_crypto = TRUE
 )

```

## Parameters

<ParamField body="discriminant_params" type="any">
  A single-row data frame from excellent discriminants
 containing columns a, b, c, discriminant, and overall_score
</ParamField>
<ParamField body="precision" type="any">
  MPFR precision in bits for calculations (default: 256)
</ParamField>
<ParamField body="use_crypto" type="any">
  Enable cryptographic mixing with ChaCha20 (default: TRUE)
</ParamField>

## Returns


 A list containing PRNG configuration:

 a Quadratic coefficient
 b Linear coefficient
 c Constant term
 precision MPFR precision in bits
 use_parallel_filling Set to FALSE for stability
 use_cryptographic_mixing ChaCha20 mixing flag



## Details


 The function creates a configuration optimized for production use:

  Parallel filling is disabled to avoid performance issues
  Cryptographic mixing is enabled by default for enhanced security
  The discriminant parameters are validated to be from excellent set


 The function also prints a summary of the configuration including the
 discriminant value and quality score.


## Examples

```r


 # Get the best discriminant and create config
 excellent <- load_excellent_discriminants()
 best_discriminant <- excellent[1, ]
 config <- create_excellent_prng_config(best_discriminant)

 # Create config with custom precision
 config_512 <- create_excellent_prng_config(best_discriminant, precision = 512)


```
