# Permutation test for hypothesis testing


 Performs a permutation test to assess the significance of a test statistic
 by comparing it to the distribution of the statistic under random
 permutations of the data.


## Usage

```r

 permutation_test(
   x,
   y = NULL,
   test_statistic,
   n_permutations = 10000,
   alternative = "two.sided",
   progress = TRUE,
   seed = NULL
 )

```

## Parameters

<ParamField body="x" type="any">
  First sample or full data vector
</ParamField>
<ParamField body="y" type="any">
  Second sample (for two-sample tests) or NULL (for one-sample tests)
</ParamField>
<ParamField body="test_statistic" type="any">
  Function that computes test statistic from data
</ParamField>
<ParamField body="n_permutations" type="any">
  Number of permutations (default: 10000)
</ParamField>
<ParamField body="alternative" type="any">
  Character string specifying the alternative hypothesis
</ParamField>
<ParamField body="progress" type="any">
  Logical; show progress bar (default: TRUE)
</ParamField>
<ParamField body="seed" type="any">
  Random seed for reproducibility (default: NULL)
</ParamField>

## Returns


 A list containing:
 p.value The permutation p-value
 permutation_stats Vector of permutation test statistics
 observed_stat The observed test statistic
 n_permutations Number of permutations used


## Examples

```r


 # Two-sample test
 x <- rnorm(50, mean = 0)
 y <- rnorm(50, mean = 0.5)
 stat_func <- function(x, y) mean(x) - mean(y)
 result <- permutation_test(x, y, stat_func)


```
