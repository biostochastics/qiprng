# Run complete analysis with Quarto reporting


 Run complete analysis with Quarto reporting


## Usage

```r

 run_complete_analysis_with_quarto(
   discriminants_file = "discriminants.csv",
   sample_size = 10000,
   max_discriminants = NULL,
   output_name = "discriminant_analysis_report"
 )

```

## Parameters

<ParamField body="discriminants_file" type="any">
  Path to discriminants CSV
</ParamField>
<ParamField body="sample_size" type="any">
  Sample size per discriminant
</ParamField>
<ParamField body="max_discriminants" type="any">
  Maximum discriminants to test (NULL for all)
</ParamField>
<ParamField body="output_name" type="any">
  Output file name prefix
</ParamField>

## Returns


 Complete analysis results with report
