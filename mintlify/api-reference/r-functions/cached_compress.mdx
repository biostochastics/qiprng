# Cached compression function


 Cached wrapper for memCompress() function. Uses R.cache to store
 results for repeated compressions with the same data and parameters.


## Usage

```r

 cached_compress(x, type = c("gzip", "bzip2", "xz"), ...)

```

## Parameters

<ParamField body="x" type="any">
  A raw vector to compress
</ParamField>
<ParamField body="type" type="any">
  Compression type
</ParamField>
<ParamField body="..." type="any">
  Additional arguments passed to memCompress()
</ParamField>

## Returns


 Compressed raw vector
