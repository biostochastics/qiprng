# Master Validation Framework


 Runs comprehensive validation of the entire qiprng testing framework

## Usage

```r


validate_qiprng_framework(level = "standard", config = NULL, verbose = TRUE)
```

## Parameters

<ParamField body="level" type="any">
  Validation level: "quick", "standard", or "comprehensive"
</ParamField>

<ParamField body="config" type="any">
  Optional configuration list
</ParamField>

<ParamField body="verbose" type="any">
  Logical indicating whether to print progress
</ParamField>

## Returns


 Validation report object
