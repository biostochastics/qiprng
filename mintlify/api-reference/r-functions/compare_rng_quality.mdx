# Compare the quality of random numbers from different generators


 This function performs a comprehensive quality assessment of different random number
 generators, applying various statistical tests and visualizations to evaluate
 randomness, uniformity, and independence properties.


## Usage

```r

 compare_rng_quality(
   n = 50000,
   generators = list(
     "qiprng" = function(n) generatePRNG(n),
     "base_r" = function(n) runif(n)
   ),
   save_plots = TRUE,
   output_dir = "quality_comparison"
 )

```

## Parameters

<ParamField body="n" type="any">
  Sample size to use for quality assessment
</ParamField>
<ParamField body="generators" type="any">
  List of generator functions to compare
</ParamField>
<ParamField body="save_plots" type="any">
  Whether to save quality comparison plots to disk
</ParamField>
<ParamField body="output_dir" type="any">
  Directory to save plots if save_plots is TRUE
</ParamField>

## Returns


 A list containing:

       metrics: Data frame with statistical quality metrics for each generator
       plots: List of ggplot objects visualizing the quality assessments
       raw_data: Raw random number samples used in the assessment



## Examples

```r


 # Compare quality between QIPRNG and R's built-in generator
 quality <- compare_rng_quality(
   n = 20000,
   generators = list(
     "qiprng" = function(n) generatePRNG(n),
     "base_r" = function(n) runif(n)
   ),
   save_plots = FALSE
 )

 # View quality metrics
 print(quality$metrics)

 # Display one of the comparison plots
 print(quality$plots$qq_plot)

```
