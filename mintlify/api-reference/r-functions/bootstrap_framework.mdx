# Bootstrap Framework for Statistical P-value Calculations


 This module provides bootstrap-based methods for calculating p-values
 and performing hypothesis tests, especially for situations where
 analytical distributions are unknown or complex.


## Usage

```r

 bootstrap_p_value(
   data,
   test_statistic,
   observed_stat = NULL,
   n_bootstraps = 10000,
   alternative = "two.sided",
   progress = TRUE,
   seed = NULL
 )

```

## Parameters

<ParamField body="data" type="any">
  Numeric vector of observations
</ParamField>
<ParamField body="test_statistic" type="any">
  Function that computes the test statistic from data
</ParamField>
<ParamField body="observed_stat" type="any">
  The observed test statistic (if NULL, will be computed)
</ParamField>
<ParamField body="n_bootstraps" type="any">
  Number of bootstrap samples (default: 10000)
</ParamField>
<ParamField body="alternative" type="any">
  Character string specifying the alternative hypothesis:
 "two.sided", "less", or "greater" (default: "two.sided")
</ParamField>
<ParamField body="progress" type="any">
  Logical; show progress bar for long computations (default: TRUE)
</ParamField>
<ParamField body="seed" type="any">
  Random seed for reproducibility (default: NULL)
</ParamField>

## Returns


 A list containing:
 p.value The bootstrap p-value
 bootstrap_stats Vector of bootstrap test statistics
 observed_stat The observed test statistic
 n_bootstraps Number of bootstrap samples used


## Details


 The framework includes:

  Bootstrap p-value calculation using resampling
  Permutation testing for non-parametric hypothesis tests
  Monte Carlo p-value estimation via simulation


 These methods are particularly useful for compression tests and other
 statistical tests where the null distribution is not well-defined.


## Examples

```r


 # Test if mean differs from 0.5
 x <- runif(100)
 stat_func <- function(d) mean(d) - 0.5
 result <- bootstrap_p_value(x, stat_func)
 print(result$p.value)


```
