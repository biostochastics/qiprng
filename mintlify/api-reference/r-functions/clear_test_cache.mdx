# Clear test result cache


 Removes cached test results for a specific category or all categories.


## Usage

```r

 clear_test_cache(category = NULL)

```

## Parameters

<ParamField body="category" type="any">
  Test category to clear (NULL for all categories)
</ParamField>

## Returns


 Invisible NULL
