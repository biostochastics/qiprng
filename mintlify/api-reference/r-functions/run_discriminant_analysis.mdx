# Run tests on all discriminants and generate report (Parallel Implementation)


 Performs comprehensive statistical testing on multiple discriminants using
 parallel processing for efficiency. Tests each discriminant with the full
 battery of randomness tests and compiles results.


## Usage

```r

 run_discriminant_analysis(
   discriminants_file = "discriminants.csv",
   sample_size = 50000,
   max_discriminants = NULL,
   n_cores = NULL,
   chunk_size = 50
 )

```

## Parameters

<ParamField body="discriminants_file" type="any">
  Path to discriminants CSV file (default: "discriminants.csv")
</ParamField>
<ParamField body="sample_size" type="any">
  Number of samples to generate per discriminant (default: 50000)
</ParamField>
<ParamField body="max_discriminants" type="any">
  Maximum number of discriminants to test, NULL for all (default: NULL)
</ParamField>
<ParamField body="n_cores" type="any">
  Number of CPU cores to use for parallel processing (default: auto-detect - 3)
</ParamField>
<ParamField body="chunk_size" type="any">
  Number of discriminants per parallel chunk for progress tracking (default: 50)
</ParamField>

## Returns


 A list containing test results for all discriminants. Each element contains:

 index Index of the discriminant in the input file
 parameters Discriminant parameters (a, b, c, discriminant)
 sample_size Number of samples tested
 uniformity Uniformity test results
 independence Independence test results
 autocorrelation Autocorrelation test results
 moments Moment analysis results
 periodicity Periodicity test results
 advanced Advanced test results
 overall_score Combined score (0-1)
 quality_rating Quality classification
 error Error message if test failed



## Details


 The function uses parallel processing to efficiently test multiple discriminants:

  Automatically detects available CPU cores
  Processes discriminants in chunks for memory efficiency
  Provides progress updates during execution
  Includes timeout protection (300s default + dynamic adjustment)
  Handles errors gracefully without stopping the entire analysis


 Runtime estimation is provided based on sample size:

  ≤50,000 samples: ~15 seconds per discriminant
  ≤100,000 samples: ~25 seconds per discriminant
  >100,000 samples: ~0.00025 * sample_size seconds



## Examples

```r


 # Test first 10 discriminants with default settings
 results <- run_discriminant_analysis(max_discriminants = 10)

 # Test all discriminants with custom settings
 results <- run_discriminant_analysis(
   sample_size = 100000,
   n_cores = 8,
   chunk_size = 100
 )

 # Analyze results
 excellent <- sapply(results, function(r) r$quality_rating == "Excellent")
 cat("Excellent discriminants:", sum(excellent), "\n")


```
