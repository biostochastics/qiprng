# Create a new PRNG instance


 Creates a new global PRNG instance with the specified configuration. This function
 initializes a new pseudo-random number generator based on quadratic irrationals with
 configurable parameters.

## Usage

```r


createPRNG(config = default_config)
```

## Parameters

<ParamField body="config" type="any">
  List of configuration parameters.
  Default values are provided by  default_config .
- Possible parameters include:

 a, b, c Integer parameters for the quadratic equation (ax² + bx + c).
  Must satisfy b² - 4ac > 0
 mpfr_precision Integer precision in bits for MPFR computations (24-10000)
 buffer_size Integer size of the buffer for random numbers
- distribution String specifying the distribution: "uniform_01", "uniform_range", "normal", "exponential", "poisson", "gamma", or "beta"
 range_min, range_max Numeric bounds for uniform_range distribution
 normal_mean, normal_sd Parameters for normal distribution
 exponential_lambda Rate parameter for exponential distribution
 poisson_lambda Rate parameter for Poisson distribution
 gamma_shape, gamma_scale Parameters for gamma distribution
 beta_alpha, beta_beta Parameters for beta distribution
- use_crypto_mixing Logical: whether to apply cryptographic mixing
- reseed_interval Integer: number of iterations between automatic reseeds
- use_threading Logical: whether to enable thread-local PRNG instances
- use_csv_discriminants Logical: whether to use custom discriminants from discriminants.csv
- use_excellent_only Logical: whether to restrict to excellent discriminants (max_abs_acf <= 0.010)
- quality_threshold Numeric: maximum autocorrelation threshold for discriminant selection
- use_parallel_filling Logical: whether to use parallel buffer filling
- mixing_strategy String: entropy mixing strategy for multi-QI ensemble (v0.5.0).
- Options: "round_robin" (default), "xor_mix", "averaging", "modular_add", "cascade_mix"
- debug Logical: whether to enable debugging output
 seed Numeric seed for reproducible sequences (0 to 2^53-1).
 When provided, all random initialization is replaced with
 deterministic seeding.
  Default NULL uses secure random initialization.
</ParamField>

## Returns


 Invisibly returns NULL
  PRNG with default settings
