# Export cached test results


 Exports all cached test results to a file for archival or sharing.


## Usage

```r

 export_cached_results(file, category = NULL)

```

## Parameters

<ParamField body="file" type="any">
  Path to export file (RDS format)
</ParamField>
<ParamField body="category" type="any">
  Optional category filter
</ParamField>

## Returns


 Invisible TRUE on success
