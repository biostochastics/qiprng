# Cached ACF function


 Cached wrapper for stats::acf() function. Uses R.cache to store
 results for repeated calls with the same data and parameters.


## Usage

```r

 cached_acf(x, lag.max = NULL, ...)

```

## Parameters

<ParamField body="x" type="any">
  A univariate or multivariate time series
</ParamField>
<ParamField body="lag.max" type="any">
  Maximum lag at which to calculate the acf
</ParamField>
<ParamField body="..." type="any">
  Additional arguments passed to acf()
</ParamField>

## Returns


 An object of class "acf"
