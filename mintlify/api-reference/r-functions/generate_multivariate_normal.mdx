# Generate Multivariate Normal Distribution


 Generates random vectors from a multivariate normal distribution using
 Cholesky decomposition.


## Usage

```r

 generate_multivariate_normal(n, mean, covariance)

```

## Parameters

<ParamField body="n" type="any">
  Number of random vectors to generate
</ParamField>
<ParamField body="mean" type="any">
  Mean vector of length d
</ParamField>
<ParamField body="covariance" type="any">
  Covariance matrix (d x d, must be positive definite)
</ParamField>

## Returns


 Matrix of n rows and d columns, where each row is a random vector


## Details


 Uses the Cholesky decomposition of the covariance matrix to
 transform independent standard normal variables into correlated
 multivariate normal variables. Requires Eigen library.


## Examples

```r


 # 2D multivariate normal
 mean <- c(0, 0)
 cov <- matrix(c(1, 0.5, 0.5, 1), 2, 2)
 x <- generate_multivariate_normal(1000, mean, cov)

 # 3D multivariate normal with correlation
 mean3 <- c(1, 2, 3)
 cov3 <- matrix(c(
   1, 0.3, 0.2,
   0.3, 1, 0.4,
   0.2, 0.4, 1
 ), 3, 3)
 y <- generate_multivariate_normal(1000, mean3, cov3)


```
