# Initialize cache directory


 Creates the cache directory if it doesn't exist and sets up
 R.cache options.


## Usage

```r

 init_cache(cache_dir = NULL, enabled = TRUE)

```

## Parameters

<ParamField body="cache_dir" type="any">
  Directory for cache storage
</ParamField>
<ParamField body="enabled" type="any">
  Whether caching is enabled
</ParamField>

## Returns


 Invisible NULL
