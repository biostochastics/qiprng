# Gap test for randomness


 Tests the distribution of gaps between occurrences of values in a specified
 range. The test analyzes how far apart values within  alpha, beta  appear
 in the sequence, which should follow a geometric distribution for truly
 random data.


## Usage

```r

 test_gaps(samples, alpha = 0.3, beta = 0.7)

```

## Parameters

<ParamField body="samples" type="any">
  Numeric vector of random samples to test
</ParamField>
<ParamField body="alpha" type="any">
  Lower bound for the gap range (default: 0.3)
</ParamField>
<ParamField body="beta" type="any">
  Upper bound for the gap range (default: 0.7)
</ParamField>

## Returns


 A list containing gap test results:

 test_name Name of the test
 statistic Chi-squared test statistic
 p_value P-value from chi-squared test
 passed Logical; TRUE if p-value > 0.05
 gaps_analyzed Number of gaps found and analyzed
 interpretation Human-readable interpretation of results



## Details


 The gap test examines the distances between consecutive occurrences of
 values within the range  alpha, beta . For a uniform random sequence,
 these gaps should follow a geometric distribution with parameter p = beta - alpha.

 The test procedure:

  Identifies all positions where samples fall within  alpha, beta
  Calculates gaps between consecutive positions
  Compares observed gap distribution to expected geometric distribution
  Uses chi-squared test to assess goodness-of-fit


 The expected frequency for gap length k is:
 E[k] = n \cdot (1-p)^{k-1} \cdot p

 where n is the number of gaps and p = beta - alpha.


## Examples

```r


 # Test gaps in uniform random numbers
 samples <- runif(10000)
 result <- test_gaps(samples)
 print(paste("Gap test p-value:", round(result$p_value, 4)))

 # Test with different range
 result <- test_gaps(samples, alpha = 0.2, beta = 0.8)


```
